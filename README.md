# Jebel Ressas Eco Frontend

A bilingual eco-tourism frontend experience for Jebel Ressas in Tunisia, built with React, Vite, and GSAP.

## Features

- Fullscreen hero with real landscape photography
- French and Arabic language switcher with RTL support
- Scroll-based storytelling powered by GSAP ScrollTrigger
- Organic green color system, paper-grain texture, and soft natural motion
- Real-image gallery with lightbox and source credits
- Nature-inspired experience cards and editorial storytelling sections
- Responsive layout for mobile, tablet, and desktop

## Image Sources

The site uses real Tunisian landscape images served from Pexels URLs. No API key is required for local development.

## Run Locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Production Build

```bash
npm run build
npm run preview
```
